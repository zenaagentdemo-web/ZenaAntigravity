# PWA Assets

This directory contains static assets for the Progressive Web App.

## Required Icons

The following icon files need to be added to this directory:

- `pwa-192x192.png` - 192x192px icon for PWA
- `pwa-512x512.png` - 512x512px icon for PWA
- `apple-touch-icon.png` - 180x180px icon for iOS
- `favicon.ico` - 32x32px favicon

These icons should feature the Zena branding and be optimized for their respective platforms.

## Current Files

- `manifest.json` - PWA manifest configuration
- `robots.txt` - Search engine crawler instructions
